# Vrinda-Store-Data-Analysis

Vrinda Store Sales Analysis Dashboard

Developed an interactive Excel dashboard to analyze 2022 e-commerce sales performance for Vrinda Store. The project includes data cleaning, transformation, Pivot Tables, Pivot Charts, and KPI visualization. Key analyses cover monthly sales trends, order status distribution, gender-wise sales contribution, age-group analysis, top-performing states, and channel-wise performance. The dashboard helps stakeholders make data-driven decisions to improve sales and customer engagement.
